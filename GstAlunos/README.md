# Gestão de Alunos

## Descrição

O projeto "Gestão de Alunos" é um sistema para gerenciar informações de alunos, permitindo operações de CRUD (Create, Read, Update, Delete). Utiliza Node.js, Express e MongoDB para fornecer uma API RESTful.

## Requisitos

- Node.js
- MongoDB
